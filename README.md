# Project Title (Update)

Game Completion is an app for android devices to track games you are currently playing and mark if you completed it or not.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install Android Studio and the Android SDK.
2. Open this project folder in Android Studio.
3. Make sure Gradle uses Java 17
4. Sync Gradle
5. Run the app

Steps to use the app:

1. Enter a Game Title
2. Enter a description
3. Check Completed if you have finished the game.
4. Tap Add Game to add it to your list.
5. To edit a game:
   - Double-tap a game card to open the Notes + Completed editor.
6. To delete a single game:
   - Tap the *rash icon on that game card.
7. To clear all games:
   - Tap the trash icon in the top bar, then confirm.
